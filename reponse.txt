Le fichier package.json sert à stocker les différents modules/frameworks que notre projet node contiendra.
Il stockera également les données de notre projet (version, description, auteur, etc...)
On peut également y assigner des script que nous pourrons appeler (ici "test" fait simplement un echo console)

Question 3 : 

le flag --save-dev permet d'ajouter directement le package dans les dépendances du projet (package.json).
Ce n'est pas une bonne idée de l'utiliser car il est utilisé en tant que dépendance de développement mais en
production ce dernier n'est pas correctement initialisé dans le projet ce qui provoquera des erreurs.
